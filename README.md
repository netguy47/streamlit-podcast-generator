# 🎙️ Streamlit Podcast Generator

A simple web app that converts text into podcasts using Microsoft Edge's text-to-speech engine.

## Features

- 🎤 Convert text to high-quality audio podcasts
- 🗣️ Multiple voice options (US and UK English)
- 💾 Download generated podcasts as MP3 files
- 🌐 Easy-to-use web interface

## How to Run Locally

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
streamlit run web_podcast.py
```

## Voices Available

- en-US-AriaNeural (US Female)
- en-US-GuyNeural (US Male)
- en-GB-SoniaNeural (UK Female)
- en-GB-RyanNeural (UK Male)

## Technologies Used

- Streamlit
- Microsoft Edge TTS (edge-tts)
